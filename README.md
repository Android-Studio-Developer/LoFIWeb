# LoFIWeb

LoFI in Your Web! Discord 없이도 웹에서 바로 LoFi를 함께 듣거나, 혼자 집중할 수 있는 **느리고 편안한 음악 + 앰비언스 사운드** 프로젝트입니다.

## 🎯 프로젝트 개요

- 🎧 **함께 듣기 / 혼자 듣기**: 음성 채팅 없이도 같은 트랙을 동기화해서 감상하거나, 개인 집중 모드로 사용.
- 🔁 **백그라운드 애니메이션**: chill & modern 톤의 부드러운 루프 애니메이션(불쾌한 골짜기 느낌 없음).
- ⌚️ **집중형 UI**: 시간 표시(컴팩트), 가독성 높은 폰트, 눈이 편한 컬러 팔레트.
- 👴🏻 **전 연령 접근성**: 연령대와 무관하게 직관적으로 사용할 수 있는 간단한 구조.
- 🔊😌 **Additional Sound**: 빗소리/카페/화이트노이즈 등 듣기 좋은 보조 사운드 믹스.
- 🎹🐌 **슬로우 템포 중심**: 시끄럽지 않고 공부/독서에 맞는 저자극 음악 큐레이션.

## 👫 같이 듣기 기능 아이디어

- 사용자는 `Join Code`로 같은 룸에 입장.
- 방장은 재생/일시정지/트랙 전환을 동기화.
- 방장이 Additional Sound(볼륨/온오프)를 🎛️ 제어.
- 룸 참여자는 동일한 타임라인으로 감상.

## 🧱 제안 기능 구조

1. **Player Core**
   - LoFi 스트림 재생
   - 재생 상태 관리 (play/pause/seek)
2. **Room Sync**
   - 룸 생성/입장 코드
   - 호스트 권한 및 동기화 이벤트
3. **Ambience Mixer**
   - 보조 사운드 레이어
   - 개별 볼륨 슬라이더
4. **Visual Scene**
   - 저자극 애니메이션 배경
   - 다크/라이트 테마
5. **Accessibility**
   - 큰 텍스트 옵션
   - 고대비 모드
   - 키보드 네비게이션
